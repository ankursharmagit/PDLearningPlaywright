# PDLearningPlaywright

Lightweight Playwright examples and setup verification for learning purposes.

## Project Layout

- `02_verify_setup.js` — simple verification script
- `Chapter_01_Basics/01_JS_Basics.js` — basic JS examples

## Prerequisites

- Node.js (14+ recommended)
- npm or yarn

## Quick start

1. Install dependencies (if you add any):

```bash
npm install
```

2. Run a script directly with Node:

```bash
node 02_verify_setup.js
```

## Notes

- This repo contains example files for Playwright learning; add a `package.json` and Playwright tests as needed.
